Description

The variable representation is orthogonal to the other definitions in the ontologies: any property can be associated with a variable and consequently also constraints, if need be. It means that not all properties of the same type need be variables; instead, this can be determined by the use scenario.

Overview

An important part of construction related information is expressed in terms of variables - properties with changing values - and constraints applying to the variables. At early stages of design and planning when detailed decisions have not been made yet, the constraints and remaining options can give valuable information. When observations of the actual entities and exectution are made, they can be related to these variables. The Digital Construction Variables ontology provides a representation of variables as objectified properties that can be associated with measurements and values at different states.
